# K-HUB Frontend - Final UI

واجهة K-HUB جاهزة كفرونت إند كامل لعرض وحجز الملاعب.

## التشغيل

```bash
npm install
npm run dev
```

افتح الرابط:

```text
http://localhost:5173
```

## أهم الصفحات

- `/` الصفحة الرئيسية
- `/courts` الملاعب
- `/courts/:courtId` تفاصيل الملعب
- `/book/:courtId` الحجز
- `/offers` العروض والأسعار
- `/about` عن النادي
- `/contact` تواصل معنا
- `/login` تسجيل الدخول
- `/register` تسجيل مستخدم جديد
- `/my-bookings` حجوزاتي
- `/admin` لوحة الإدارة
- `/schedule` جدول المواعيد

## خصائص الواجهة

- تصميم متوافق مع هوية K-HUB وألوان اللوجو.
- تسجيل مستخدم جديد وتسجيل دخول بإيميل وباسورد.
- صلاحيات مستخدم عادي ومسؤول.
- اختيار الملعب، التاريخ، وقت البداية، وقت النهاية.
- مواعيد العمل من 6 مساءً إلى 4 فجرًا.
- منع الحجز عند وجود تداخل مع مدة محجوزة.
- اختيار طريقة الدفع.
- رفع إثبات الدفع عند الدفع الإلكتروني.
- مراجعة إثبات الدفع من لوحة الإدارة.
- تأكيد أو رفض الحجز من الإدارة.
- عرض حالة الحجز للعميل.
- دعم اللغة العربية والإنجليزية والفرنسية.
